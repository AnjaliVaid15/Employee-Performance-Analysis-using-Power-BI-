*This project analyzes employee HR data using Excel and Power BI to generate key insights through an interactive dashboard*

## Objectives
1 Analyze employee performance and salary data
2 Apply commonly used Excel functions
3 Create an interactive HR analytics dashboard in Power BI
4 Present key HR insights in a simple and visual manner

## Files Included
1 employee data.xlsx – Raw dataset used for analysis
2 HR analysis dashboard.pbix – Power BI dashboard
3 dashboard image – Screenshot of the dashboard (added in case the PBIX file does not open)

## Tools Used
1 Microsoft Excel
2 Microsoft Power BI

## Key Insights
1 Total Employees
2 Average Salary
3 Department-wise Analysis
4 Experience & Salary Trends

## How to Use
1 Open the Power BI file to explore the dashboard.
2 If the dashboard does not open, refer to the dashboard image for a quick overview.
